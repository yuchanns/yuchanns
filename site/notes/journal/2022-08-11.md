* 删除乱码文件名 [[Linux|linux]]
    * 使用 inode 删除：
      ```bash
      ls -i . # 获取文件节点号
      find -inum 节点号 -delete
      ```
* swaybg 随机背景 [[M1 Sway 全纪实|posts/sway-on-m1-full-record]]
    * 600s 随机 [来源]()
      ```bash
      #!/usr/bin/env sh
      while true; do
        PID=`pidof swaybg`
        swaybg -i $(find $HOME/Backgrounds/. -type f | shuf -n1) -m fill &
        sleep 1
        kill $PID
        sleep 599
      done
      ```
